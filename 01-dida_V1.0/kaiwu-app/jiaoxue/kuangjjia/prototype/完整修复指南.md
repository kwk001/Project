### 步骤2：添加JavaScript函数

在 `<script>` 标签内，第一个函数定义前添加：

```javascript
// 全选/取消全选
function toggleSelectAll(type) {
    const selectAllCheckbox = document.getElementById(`selectAll${type.charAt(0).toUpperCase() + type.slice(1)}`);
    const checkboxes = document.querySelectorAll(`.${type}-checkbox`);
    checkboxes.forEach(cb => cb.checked = selectAllCheckbox.checked);
}

// 切换每页显示条数
function changePageSize(type) {
    const pageSize = document.getElementById(`${type}PageSize`).value;
    console.log(`切换${type}每页显示条数为: ${pageSize}`);
    // 这里可以添加重新渲染表格的逻辑
}

// 树节点展开/收起（带自适应）
function toggleNode(element) {
    const toggle = element.querySelector('.tree-toggle');
    const children = element.nextElementSibling;
    
    if (toggle && children) {
        toggle.classList.toggle('expanded');
        children.classList.toggle('collapsed');
        
        // 触发自适应调整
        setTimeout(() => {
            adjustLayout();
        }, 300);
    }
}

// 页面布局自适应调整
function adjustLayout() {
    // 重新计算内容区域高度
    const contentArea = document.querySelector('.content-area');
    if (contentArea) {
        contentArea.style.height = 'auto';
    }
}
```

## 六、常见问题

### Q1: 多选框点击没有反应？
**A:** 检查JavaScript函数是否正确添加，特别是 `toggleSelectAll` 函数。

### Q2: 分页组件样式错乱？
**A:** 检查CSS样式是否正确添加，特别是 `.pagination-size` 和 `.pagination-jump`。

### Q3: 树状图展开后内容没有自适应？
**A:** 确保 `toggleNode` 函数中调用了 `adjustLayout()`，并且有 `setTimeout` 延迟。

### Q4: 脚本运行报错？
**A:** 检查Python版本是否为3.6+，文件路径是否正确。
