# 古麒绒材溯源管理系统

## 项目简介

安徽古麒绒材股份有限公司（股票代码：001390）产品溯源管理系统，实现产品从生产到销售的全流程追溯。

**核心业务**：
- 产品质量可追溯（二维码技术）
- 消费者信任建立（透明信息查询）
- 品牌价值提升（数字化形象）
- 防伪打假（唯一溯源码）

## 快速开始

### 环境要求

- Node.js 16.x+
- MongoDB 4.4+
- @kaiwu-cloud/lowcode-engine 1.1.2+

### 安装部署

```bash
# 1. 克隆项目
cd /opt/guqirongcai

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 构建生产环境
npm run build
```
