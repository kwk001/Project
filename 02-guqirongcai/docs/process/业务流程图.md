## 1. 系统整体业务流程

```mermaid
flowchart TD
    Start([开始]) --> Init[初始化基础数据]
    Init --> BaseData{基础数据维护}

    BaseData -->|计量单位| Unit[计量单位管理]
    BaseData -->|物料分类| Category[物料分类管理]
    BaseData -->|物料档案| Material[物料档案管理]
    BaseData -->|客户档案| Customer[客户档案管理]

    Unit --> PrintFlow[产品码打印流程]
    Category --> PrintFlow
    Material --> PrintFlow

    PrintFlow --> Generate[生成产品码]
    Generate --> Print[打印溯源码]
    Print --> Record[记录打印历史]

    Record --> Consumer[消费者扫码]
    Consumer --> Scan[微信扫码]
    Scan --> Display[H5展示溯源信息]

    Display --> End([结束])
```

## 3. 产品码打印流程

```mermaid
flowchart TD
    Start([开始]) --> SelectMaterial[选择物料]
    SelectMaterial --> InputBatch[填写批次号]
    InputBatch --> InputQuantity[填写数量]
    InputQuantity --> InputDate[选择生产日期]
    InputDate --> InputQuality[填写质检信息]

    InputQuality --> Validate{数据校验}
    Validate -->|校验失败| InputBatch
    Validate -->|校验通过| GenerateCode[生成产品码]

    GenerateCode --> Preview[预览二维码]
    Preview --> ConfirmPrint{确认打印?}

    ConfirmPrint -->|取消| Modify[修改信息]
    Modify --> InputBatch

    ConfirmPrint -->|确认| CheckDevice{检查打印设备}
    CheckDevice -->|设备异常| DeviceError[提示设备错误]
    DeviceError --> List

    CheckDevice -->|设备正常| Print[执行打印]
    Print --> Record[保存打印记录]
    Record --> UpdateStatus[更新打印状态]
    UpdateStatus --> List[返回列表]

    List --> End([结束])
```

## 5. 微信扫码溯源流程

### 5.1 消费者扫码流程

```mermaid
flowchart TD
    Start([消费者扫码]) --> WechatScan[微信扫描二维码]
    WechatScan --> ParseCode[解析二维码内容]
    ParseCode --> ValidateCode{校验码有效性}

    ValidateCode -->|无效码| Invalid[显示无效码提示]
    Invalid --> End1([结束])

    ValidateCode -->|已过期| Expired[显示过期提示]
    Expired --> End2([结束])

    ValidateCode -->|有效码| LoadData[加载溯源数据]
    LoadData --> GetProduct[获取产品信息]
    GetProduct --> GetQuality[获取质检信息]
    GetQuality --> GetFactory[获取企业信息]
    GetFactory --> GetTrace[获取溯源流程]

    GetTrace --> RenderPage[渲染H5页面]
    RenderPage --> ShowResult[展示验证结果]
    ShowResult --> ShowProduct[展示产品信息]
    ShowProduct --> ShowQuality[展示质检指标]
    ShowQuality --> ShowFactory[展示企业信息]
    ShowFactory --> ShowTrace[展示溯源流程]

    ShowTrace --> RecordQuery[记录查询日志]
    RecordQuery --> End3([结束])
```

## 8. 异常处理流程

### 8.1 打印异常处理

```mermaid
flowchart TD
    Start([开始打印]) --> SendCommand[发送打印指令]
    SendCommand --> CheckResult{检查结果}

    CheckResult -->|成功| Record[记录成功日志]
    Record --> End1([结束])

    CheckResult -->|设备离线| OfflineError[提示设备离线]
    OfflineError --> Retry{重试?}
    Retry -->|是| SendCommand
    Retry -->|否| Manual[转人工处理]
    Manual --> End2([结束])

    CheckResult -->|纸张不足| PaperError[提示缺纸]
    PaperError --> WaitAdd[等待加纸]
    WaitAdd --> SendCommand

    CheckResult -->|打印失败| PrintError[提示打印失败]
    PrintError --> RecordFail[记录失败日志]
    RecordFail --> Manual
```

### 8.2 扫码异常处理

```mermaid
flowchart TD
    Start([扫码]) --> Parse{解析结果}

    Parse -->|无法识别| InvalidCode[显示：无效二维码]
    InvalidCode --> Suggest1[提示：请扫描产品溯源码]
    Suggest1 --> End1([结束])

    Parse -->|码已失效| ExpiredCode[显示：溯源码已过期]
    ExpiredCode --> Suggest2[提示：请联系厂商核实]
    Suggest2 --> End2([结束])

    Parse -->|系统异常| SystemError[显示：系统繁忙]
    SystemError --> Retry{重试?}
    Retry -->|是| Parse
    Retry -->|否| Contact[提示：请稍后重试]
    Contact --> End3([结束])

    Parse -->|网络异常| NetworkError[显示：网络连接失败]
    NetworkError --> CheckNetwork{检查网络}
    CheckNetwork -->|已恢复| Parse
    CheckNetwork -->|仍异常| OfflineMode[进入离线提示]
    OfflineMode --> End4([结束])
```
