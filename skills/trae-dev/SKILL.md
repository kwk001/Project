# Trae Development Skill

## Description
启动 Trae IDE 进行开发。

## Commands
- `trae start`: 启动 Trae IDE
- `trae open <project>`: 打开指定项目
- `trae file open <file_path>`: 打开指定文件
- `trae file list <directory>`: 列出目录中的文件
- `trae file read <file_path>`: 读取文件内容
